Project: My WordPress Plugin “Shantz WP Prefix Suffix” Updated to 1.0.4

2008-10-15Shantanu Goel
#1.0.4  #PHP  #plugin  #shantz-wp-prefix-suffix  #wordpress 

For those who don’t know Shantz-WP-Prefix-Suffix is a WordPress plugin that allows you to add any text and/or HTML/CSS/PHP/Javascript code to the top or bottom of your posts (e.g. see my site links at top of each post and the copyright link and permalink at the bottom of each) I’ve updated it to version 1.0.4 which allows you to add PHP code to the top or bottom of your posts now. This is very useful for you especially if you want to display some ads that use php or javascript (I’m using it as well for adsense on this blog)
